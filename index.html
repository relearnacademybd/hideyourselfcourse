// COURSE CURRICULUM

১৩টি সম্পূর্ণ মডিউল

প্রতিটি মডিউলে ক্লিক করুন সব সাবটপিক দেখতে।

01
অনলাইন প্রাইভেসি ও সাইবার সিকিউরিটির পরিচিতি
▾
  • অনলাইন প্রাইভেসি কী এবং কেন গুরুত্বপূর্ণ
  • সাইবার সিকিউরিটির মৌলিক ধারণা
  • ডিজিটাল ফুটপ্রিন্ট — আপনি কতটা ট্র্যাক হচ্ছেন
  • অনলাইন ট্র্যাকিং কিভাবে কাজ করে
  • বর্তমান সাইবার ঝুঁকি ও হুমকি পরিচিতি
02
ল্যাব সেটআপ ও সিকিউরিটি টুলস
▾
  • নিরাপদ ল্যাব পরিবেশ তৈরি করা
  • প্রয়োজনীয় টুলস ইনস্টলেশন গাইড
  • সিস্টেম কনফিগারেশন সেটআপ
  • সাধারণ সমস্যা ও সমাধান
03
সিকিউরিটি টুলস সেটআপ ও কনফিগারেশন
▾
  • বিভিন্ন সিকিউরিটি টুলস পরিচিতি
  • টুলস কনফিগারেশন ও অপটিমাইজেশন
  • নিরাপদ ব্যবহারের পদ্ধতি
  • ত্রুটি শনাক্ত ও সমাধানের কৌশল
04
অনলাইন অ্যানোনিমিটি ও নিরাপদ ব্রাউজিং
▾
  • ডিজিটাল পরিচয় লুকানোর কৌশল
  • অনলাইন ট্র্যাকিং থেকে বাঁচার উপায়
  • ব্যক্তিগত তথ্য সুরক্ষার পদ্ধতি
  • নিরাপদ ব্রাউজিং অভ্যাস গঠন
  • ব্রাউজার প্রাইভেসি সেটিংস অপটিমাইজেশন
05
Tor ও Proxychains
▾
  • Tor নেটওয়ার্ক কী এবং কিভাবে কাজ করে
  • Tor ব্রাউজার ইনস্টলেশন ও কনফিগারেশন
  • Proxychains সেটআপ ও কনফিগারেশন
  • Tor + Proxychains একসাথে ব্যবহার
  • সাধারণ সমস্যা ও তাদের সমাধান
06
MAC Address ও ডিভাইস পরিচয় সুরক্ষা
▾
  • MAC Address কী এবং এর গুরুত্ব
  • MAC Address কিভাবে ট্র্যাকিং এ ব্যবহৃত হয়
  • ডিভাইস পরিচয় সুরক্ষার ধারণা
  • MAC Address পরিবর্তন ও কনফিগারেশন
  • প্র্যাকটিক্যাল সেশন — হাতে-কলমে অনুশীলন
07
Anti-Detect Browser ও Browser Fingerprint Protection
▾
  • Anti-Detect Browser কী এবং কেন প্রয়োজন
  • Browser Fingerprint কী — কিভাবে আপনি শনাক্ত হন
  • Fingerprint Tracking কিভাবে কাজ করে
  • Browser Tracking Methods বিশ্লেষণ
  • Browser Profile Management
  • Multiple Profile Configuration
  • Cookie Isolation কৌশল
  • Fingerprint Leak Detection
  • Anti Tracking Configuration
  • Privacy Protection Techniques
08
Android Security Masterclass
▾
  • Android Security Fundamentals
  • App Permission Analysis — কোন অ্যাপ কী নেয়
  • ক্ষতিকর অ্যাপ শনাক্তকরণের পদ্ধতি
  • নিরাপদ অ্যাপ ব্যবহারের নীতি
  • Android Privacy Settings অপটিমাইজেশন
  • Device Hardening Techniques
09
Windows Security Masterclass
▾
  • Windows Security Fundamentals
  • Windows Defender কনফিগারেশন ও অপটিমাইজেশন
  • Firewall Configuration — সঠিক নিয়মে সেটআপ
  • Privacy Settings Optimization
  • Secure Software Installation গাইড
  • Windows Hardening Techniques
10
Malware Detection & Analysis
▾
  • Malware কী এবং এর ধরনসমূহ
  • Virus, Trojan, Worm, Spyware, Ransomware পরিচিতি
  • Malware কিভাবে ছড়ায় — বিস্তারের পথ
  • Malware কিভাবে কাজ করে — কার্যপ্রণালী
  • আক্রান্ত ডিভাইসের লক্ষণ চেনার উপায়
  • সন্দেহজনক ফাইল শনাক্তকরণ
  • নিরাপদ বিশ্লেষণ পদ্ধতি
11
Malware Removal & System Cleanup
▾
  • Malware শনাক্তকরণের ধাপসমূহ
  • Windows থেকে Malware অপসারণ
  • Android থেকে Malware অপসারণ
  • Browser Cleanup — দূষিত এক্সটেনশন ও সেটিংস পরিষ্কার
  • Startup Program Analysis
  • System Cleanup সম্পূর্ণ প্রক্রিয়া
  • Recovery Techniques — আক্রান্তের পর পুনরুদ্ধার
12
বাংলাদেশ সাইবার আইন ও ডিজিটাল নিরাপত্তা
▾
  • বাংলাদেশে সাইবার অপরাধের পরিচিতি
  • প্রচলিত সাইবার আইন ও বিধিমালা
  • ডিজিটাল নিরাপত্তা আইনের মূল বিষয়
  • অনলাইন প্রতারণা ও স্ক্যাম থেকে সুরক্ষা
  • সোশ্যাল মিডিয়া ব্যবহারের আইনগত দিক
  • সাইবার অপরাধ রিপোর্টিং প্রক্রিয়া
  • ডিজিটাল নাগরিকের দায়িত্ব ও অধিকার
13
বাস্তব জীবনের সাইবার সুরক্ষা
▾
  • শক্তিশালী পাসওয়ার্ড তৈরির নিয়ম
  • Two Factor Authentication (2FA) সেটআপ
  • Phishing শনাক্তকরণ ও প্রতিরোধ
  • Social Engineering Awareness
  • নিরাপদ ইমেইল ব্যবহারের পদ্ধতি
  • নিরাপদ সোশ্যাল মিডিয়া ব্যবহার
  • ব্যক্তিগত তথ্য সুরক্ষার চূড়ান্ত গাইড
